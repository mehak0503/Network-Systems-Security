# Network-Systems-Security

Assignments of the course on Network Systems and Security in IIT Delhi.

- Assignment 1 :- Implementation of **DES Algorithm** from scratch.
- Assignment 2 :- Implementation of **Public Key Distribution Authority**.
- Assignment 3 :- Implemetation of License verification System using **Digital Signatures**.
- Assignment 4 :- Implementation of **Elgamal cryptosystem**.
